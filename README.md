### Main Technologies Used

- Node - v20.15.1
- Cucumber
- Selenium

## Deployment

you need to install dependencies

```bash
  npm install
```


## Running Tests

To run code tests, run the following commands

```bash
  npm run check:code
  npm run check:features
  npm run test:jest
```

### Cucumber
To run UI tests, run the following command (working)

```bash
  npm run test:ui
```
